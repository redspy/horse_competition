# 경마 게임 기능 워크스루

모든 구현이 완료되었습니다. 무지개 색상의 말들을 선택하고 경주를 즐길 수 있습니다.

## 주요 기능
*   **시작 화면**: 참가할 말의 수(2~7)와 각 말의 색상을 설정할 수 있습니다.
*   **에셋**: `horse_base.png`(흑백)를 기반으로 파이썬 스크립트가 7가지 색상을 자동으로 입혀 생성한 일관된 스프라이트를 사용합니다. (100% 동일한 애니메이션 보장)
*   **경기 진행**: 0.5초마다 랜덤하게 이동하며, 하위권 말에게는 역전 기회(부스트)가 주어집니다.

## 테스트 결과 (자동화 테스트)
브라우저 자동화 도구를 사용하여 다음 시나리오를 검증했습니다.

1.  **게임 설정**: 말 수 5마리로 변경, 색상 변경 테스트 완료.
2.  **게임 진행**: 시작 버튼 클릭 후 카운트다운 및 레이스 진행 확인 (약 20초 소요).
3.  **결과 확인**: 레이스 종료 후 순위 모달 표시 확인.

## 파일 처리 내역
*   `process_assets.py`: 생성된 마젠타 배경의 스프라이트를 투명 배경의 PNG로 변환하여 `assets/` 폴더에 저장했습니다.
*   `assets/`: 트랙 배경 및 7가지 색상의 말 스프라이트가 포함되어 있습니다.
