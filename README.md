# 🌈 무지개 경마 게임 (Rainbow Horse Racing)

웹 브라우저에서 즐길 수 있는 간단하고 재미있는 경마 게임입니다. 무지개 색상의 말들을 선택하여 어떤 말이 1등을 할지 내기해보세요!

## 🎮 게임 특징

*   **커스터마이징**: 2마리부터 7마리까지 참가 말 수를 자유롭게 설정할 수 있습니다.
*   **다채로운 색상**: 각 말을 무지개(빨, 주, 노, 초, 파, 남, 보) 색상 중 하나로 설정할 수 있습니다.
*   **역전의 드라마**: 하위권 말에게는 부스트 확률이 적용되어 언제든지 순위가 뒤바뀔 수 있는 흥미진진한 경주가 펼쳐집니다.
*   **귀여운 그래픽**: 픽셀 아트 스타일의 말 캐릭터와 애니메이션을 제공합니다.

## 🚀 실행 방법

이 프로젝트는 별도의 서버 설치가 필요 없는 정적 웹 페이지입니다.

1.  `index.html` 파일을 웹 브라우저(Chrome, Edge 등)에서 직접 엽니다.
2.  설정 화면에서 말 수와 색상을 선택합니다.
3.  **"경기 시작!"** 버튼을 눌러 경주를 시작합니다.

## 📂 파일 구조

```
horse_competition/
├── index.html          # 게임의 메인 시작 페이지 (구조)
├── style.css           # 게임 디자인 및 애니메이션 스타일
├── app.js              # 게임 로직 (이동, 순위 계산, UI 제어)
├── process_assets.py   # 스프라이트 생성 및 색상 자동화 스크립트 (Python)
├── task.md             # 개발 작업 내역
├── walkthrough.md      # 기능 명세 및 테스트 결과
└── assets/             # 게임 리소스 폴더
    ├── track_bg.png    # 경기장 배경
    ├── horse_base.png  # 말 베이스 스프라이트 (흑백)
    └── horse_*.png     # 색상별 말 스프라이트 (자동 생성됨)
```

## 🛠️ 개발 정보

*   **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
*   **Asset Processing**: Python (Pillow 라이브러리)
    *   `process_assets.py`를 실행하면 `assets/horse_base.png`를 기반으로 다양한 색상의 말 스프라이트를 자동으로 생성합니다.
